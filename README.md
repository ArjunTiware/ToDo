Getting Started

First, install node_modules:

1.npm install

2.npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev

Open http://localhost:5173 with your browser to see the result.

You can start editing the page by modifying App.jsx. The page auto-updates as you edit the file.
